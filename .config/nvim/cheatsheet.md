# Basics
## Normal Mode
### save file: `<leader>w`
### save+quit: `<leader>q`
### quit all: `<leader>Q`

# General
## Normal Mode
- paste non-linewise text above/below current cursor
  - above: `<leader>p`
  - below: `<leader>P`

## Insert Mode
- current word -> UPPER - `<C-u>`
- current word -> Title - `<C-t>`

# LSP

## Normal Mode
- go to definition
  - `<C-]>`
  - `gd`
